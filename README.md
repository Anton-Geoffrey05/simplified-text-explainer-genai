# Simplified Legal / Medical Text Explainer (GenAI)

## Problem Statement
Legal and medical documents contain complex terminology that is difficult for non-experts to understand.

## Solution
This project uses a Large Language Model (LLM) to:
- Simplify complex text
- Extract key points
- Highlight warnings

## Tech Stack
- Python
- OpenAI LLM
- Streamlit

## How to Run
1. Clone the repo
2. Install dependencies
3. Add API key
4. Run `streamlit run app.py`

## Use Cases
- Legal contracts
- Medical reports
- Insurance documents
.
